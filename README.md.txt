# Plan Detector

Librería en Python para la detección de líneas, esquinas e intersecciones en planos arquitectónicos digitalizados.

## Características
- Preprocesamiento de imágenes
- Conversión a escala de grises
- Umbralización
- Filtrado espacial
- Corrección por transformación afín
- Detección de bordes
- Detección de esquinas
- Transformada de Hough
- Visualización de resultados

## Requisitos
pip install -r requirements.txt

## Ejecución
python -m plan_detector.MAINCHINGON

## Imagen de prueba
El proyecto incluye imágenes de prueba dentro de la carpeta IMAGENES, en el main se usa la imagen numero 5 :).
